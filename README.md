# 🍛 Indian Food Data Analysis

## 📌 Objective
Analyze Indian food dataset to extract insights using Python (NumPy & Pandas).

## 🧹 Data Cleaning
- Handled missing values
- Replaced invalid (-1) values
- Standardized text
- Removed duplicates

## ⚙️ Feature Engineering
- Created total_time column
- Categorized dishes (Quick/Medium/Slow)
- Identified high-time dishes

## 📊 Key Insights
- Most dishes belong to [Region]
- Majority dishes are Quick/Medium
- [Dish Name] takes maximum time
- Veg vs Non-veg comparison done

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## 📈 Conclusion
Dataset shows majority dishes are quick to prepare, indicating suitability for fast-service environments.
